# BC-21-Twitter-Sentiment-Analysis
A console app that use the Twitter API to fetch tweets for a particular user and save them in JSON file (for a given period e.g. 1 week or 1 month depending on the volume). Shows % progress as the Twitter API is fetching the tweets.  Perform a word-frequency analysis and list the words with their frequency starting from the highest to the lowest excluding stop-words. Perform sentiment analysis using the Alchemy API
